🚀 COMO COMPILAR

Opção 1 - RosBE (Recomendado):
  $ cd /c/ReactOS/reactos
  $ bash build_reactos.sh -m msvc -a amd64

Opção 2 - Ninja:
  $ ninja base/system/logonui/logonui

Opção 3 - Visual Studio:
  $ cmake -G "Visual Studio 16 2019" .
  $ cmake --build . --target logonui

  Mude:  #define USE_MODERN_DESIGN 1
  Para:  #define USE_MODERN_DESIGN 0

Customizar cores:
  Edite: base/system/logonui/NT6design.c linhas 15-22
  Modifique os valores RGB conforme desejado

📋 FUNCIONALIDADES IMPLEMENTADAS

Função NT6_DrawBaseBackground():
  ├─ Cria memory DC
  ├─ Desenha gradiente Aero Blue (3 camadas)
  ├─ Adiciona glass frame border
  ├─ Posiciona ícone do sistema
  └─ Copia para desktop (BitBlt)

Função NT6_CreateLogoffScreen():
  ├─ Cria font Segoe UI 28pt
  ├─ Calcula position de texto centralizado
  ├─ Desenha drop shadow (offset 2px)
  ├─ Renderiza texto anti-aliased
  └─ Atualiza tela em tempo real
