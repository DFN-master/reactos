🚀 NEXT STEPS (HOW TO BUILD)

Select one of these compilation methods:

EASIEST (If you have RosBE installed):
  $ cd E:\ReactOS\reactos
  $ bash build_reactos.sh -m msvc -a amd64
  $ ninja -C output-VS-amd64 bootcd

AUTOMATIC SCRIPT:
  → See COMPILATION_INSTRUCTIONS.md for pre-built batch script
  → Run build-admin-page.bat which handles everything

MANUAL COMMAND LINE:
  1. Setup Visual Studio environment (vcvars64.bat)
  2. Run: cmake -G Ninja -B output-VS-amd64 .
  3. Run: cd output-VS-amd64 && ninja base/setup/reactos/reactos.exe
  4. Run: ninja bootcd
  5. Result: bootcd.iso (~90 MB) with admin page integrated

DOCKER (Cleanest option):
  $ docker run --rm -v E:\ReactOS\reactos:/reactos \ 
    osrtos/reactos-build:latest \
    bash -c "ninja -C output-VS-amd64 base/setup/reactos/reactos.exe && ninja -C output-VS-amd64 bootcd"
